Install SonarQube
1) Download  latest version of sonarQube from  http://www.sonarqube.org/downloads/  
2) extract to the folder  
3) cd to SonarQube_HOME/bin/macosx-universal-64/ 
4) ./sonar.sh start
5) acceess the dashboard on http://localhost:9000

Install Javascript Plug-in( http://docs.codehaus.org/display/SONAR/Update+Center)
Login with admin/admin on sonarQube Dashboard( http://localhost:9000)
navigate to ->  settings  -->Update Center 
Verify Updated Plugins / Installed Plugins
